# 首页场景卡片角度参数设计

## 概述

本文档记录首页场景卡片旋转角度的设计逻辑和参数配置。

## 文件位置

- **代码实现**: `miniprogram/pages/index/index.js`
- **函数**: `loadScenesForTab(tabIndex)`

## 角度参数配置

### 旋转角度池

```javascript
const rotations = [-3, -3, -3, 3, 3, 3, 3, -3, -3, -3, -3, 3, 3, 3, 3, -3];
```

### 角度分配表

每行包含2个卡片（左、右），共8行16个卡片的角度配置：

| 行数 | 左卡片 | 右卡片 | 说明 |
|------|--------|--------|------|
| 第1行 | -3° | -3° | 双左倾 |
| 第2行 | -3° | +3° | 左倾右仰 |
| 第3行 | +3° | +3° | 双右仰 |
| 第4行 | +3° | -3° | 右仰左倾 |
| 第5行 | -3° | -3° | 同第1行（循环） |
| 第6行 | -3° | +3° | 同第2行（循环） |
| 第7行 | +3° | +3° | 同第3行（循环） |
| 第8行 | +3° | -3° | 同第4行（循环） |

### 设计模式

角度参数采用 **4行一循环** 的设计模式：

- **第1-4行**: 基础模式
  - 第1行: 双卡片左倾 (-3°, -3°)
  - 第2行: 左倾右仰 (-3°, +3°)
  - 第3行: 双卡片右仰 (+3°, +3°)
  - 第4行: 右仰左倾 (+3°, -3°)

- **第5-8行**: 循环复用第1-4行的模式
  - 第5行 = 第1行
  - 第6行 = 第2行
  - 第7行 = 第3行
  - 第8行 = 第4行

## 代码实现

```javascript
loadScenesForTab(tabIndex) {
  const currentCategory = this.data.tabs[tabIndex];
  if (!currentCategory) return;

  const rawScenes = getScenesByCategory(currentCategory.id);
  
  // 旋转角度池：每8个一组循环，对应4行基础模式 + 4行循环模式
  // 第一行: 左 -3°, 右 -3°
  // 第二行: 左 -3°, 右 +3°
  // 第三行: 左 +3°, 右 +3°
  // 第四行: 左 +3°, 右 -3°
  // 第五行(同第一行): 左 -3°, 右 -3°
  // 第六行(同第二行): 左 -3°, 右 +3°
  // 第七行(同第三行): 左 +3°, 右 +3°
  // 第八行(同第四行): 左 +3°, 右 -3°
  const rotations = [-3, -3, -3, 3, 3, 3, 3, -3, -3, -3, -3, 3, 3, 3, 3, -3];

  // 根据分类ID生成该分类专属的颜色排序
  const categoryColors = this.getShuffledArray(this.data.cardColors, currentCategory.id);

  // 为每个卡片分配颜色和旋转角度
  const coloredScenes = rawScenes.map((item, index) => {
    return {
      ...item,
      bgColor: categoryColors[index % categoryColors.length],
      rotate: rotations[index % rotations.length]
    };
  });

  this.setData({
    sceneList: coloredScenes
  });
}
```

## 样式应用

角度通过内联样式动态绑定到卡片元素：

```html
<view 
  class="grid-card" 
  style="background-color: {{item.bgColor}}; transform: rotate({{item.rotate}}deg);" 
  bindtap="handleSceneTap" 
  data-id="{{item.id}}"
>
  <!-- 卡片内容 -->
</view>
```

## 设计意图

1. **视觉层次感**: 通过不同的旋转角度创造错落有致的视觉效果
2. **循环一致性**: 4行一循环的模式确保视觉规律性的同时保持变化
3. **扩展性**: 16个角度的配置支持最多16个场景卡片的展示
4. **微调范围**: ±3°的小角度旋转既增加了趣味性，又不会过度影响可读性
