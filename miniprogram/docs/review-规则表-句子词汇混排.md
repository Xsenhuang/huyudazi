# 复习规则表（句子 + 词汇混排，10 任务/轮）

## 目标
| 项 | 规则 |
|---|---|
| 复习目标 | 学完场景后，通过间隔复习巩固：词汇识别、句子听懂、句子跟读、句子输出（引导式说一句） |
| 复习形态 | 系统自动组合，跨场景混排；用户完成一轮后选择继续或结束 |
| 最小复习单位 | 任务卡（ReviewTask）：句子卡（SentenceTask）或词汇卡（VocabTask） |

## 一轮复习（Pack）
| 项 | 规则 |
|---|---|
| 一轮数量 | 固定 10 张任务卡（句子卡/词汇卡都算 1） |
| 完成定义 | 做完 10 张任务卡即完成一轮 |
| 继续机制 | 完成后可选择“继续下一轮”或“结束复习” |
| 场景策略 | 跨场景混合优先，以到期优先为准 |
| 去重 | 同一轮不重复同一句/同一词；同一会话内尽量避免相邻两轮重复（若刚标“忘了”允许快速再见） |

## 对象与排期（遗忘曲线）
| 对象 | 是否独立排期 | 更新触发 | 结果口径 |
|---|---:|---|---|
| 句子（Sentence） | 是 | 完成句子卡并给出评价 | 主观三档：记住/模糊/忘了 |
| 词汇（Vocab） | 是 | 完成词汇卡并判定对错 | 对/错折算为统一口径（对≈记住，错≈忘了；可选扩展“模糊”） |
| 关键一致性 | - | 句子中出现某词 ≠ 自动复习该词；做了词汇卡 ≠ 自动更新包含它的句子 | 避免数据互相抬升导致排期失真 |

## 自动选题（生成 10 张任务卡）
| 步骤 | 规则 |
|---|---|
| 候选池 | 到期句子 + 到期词汇（dueAt <= now），各自按 dueAt 从早到晚排序 |
| 填充优先级 | 到期 > 弱项（近期出现“模糊/忘了”或错误率高） > 临近到期 |
| 关联增强 | 选入句子卡时，将该句关键词中“到期/弱”的词汇优先加入候选，以提升穿插自然度 |
| 比例约束（建议） | 每轮至少包含一定数量的句子卡与词汇卡（例如：句子 ≥ 4、词汇 ≥ 3；其余随内容池决定） |
| 兜底 | 当到期/弱项不足以凑满 10：使用“已学过内容”补齐（仍遵守去重） |

## 任务卡规则
### 词汇卡（VocabTask：识别题）
| 项 | 规则 |
|---|---|
| 题型 | 展示上海话词/短语 → 多选一选择意思 |
| 选项数 | 默认 4 选 1（1 正确 + 3 干扰）；不足时降为 3 选 1 |
| 判定 | 客观对/错 |
| 干扰项策略 | 随机但带约束（见下表） |

### 句子卡（SentenceTask）
| 项 | 规则 |
|---|---|
| 必做步骤 | 听懂句子（播放音频/理解） |
| 可选步骤（系统决定） | 跟读句子（中风险/高风险句） |
| 可选步骤（少量配额） | 引导式说一句（高风险/核心句；每轮设上限） |
| 收口评价 | 主观三档：记住/模糊/忘了 |

## 引导式说一句（输出任务）
| 项 | 规则 |
|---|---|
| 形态 | 给中文意思/场景提示/关键词（可选给句首提示），用户说出对应上海话句子 |
| 评估 | 不做机器评分；用主观三档收口更新句子排期 |
| 配额（建议常量） | 每轮最多触发 K 句（例如 1～2 句或 20%） |
| 触发优先级 | 高风险句 > 核心句 > 新学不久句 |

## 干扰项生成（随机但带约束）
| 项 | 规则 |
|---|---|
| 来源池 | 优先从“用户已学过词汇”中随机抽取（避免出现未学选项） |
| 去重/排除 | 干扰项与正确项的意思不能重复或高度相似；四个选项意思不重复 |
| 难度保护 | 干扰项的解释长度/类型尽量接近正确项（避免一眼排除） |
| 会话去重 | 同一轮内尽量不重复相同干扰项组合 |
| 兜底扩展 | 已学词池过小：从“全局已学词”扩展抽取（仍不使用未学词） |

## 常量建议（用于稳定体验）
| 常量 | 含义 | 建议默认 |
|---|---|---|
| PACK_SIZE | 每轮任务卡数量 | 10 |
| SPEAK_QUOTA_PER_PACK | 每轮“引导式说一句”上限 | 1～2 |
| MIN_SENTENCE_PER_PACK | 每轮最少句子卡数量 | 4 |
| MIN_VOCAB_PER_PACK | 每轮最少词汇卡数量 | 3 |
